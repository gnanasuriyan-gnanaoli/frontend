SEMANTIC MARKUP

These are text elements that add extra information to the page

emphasis tag emphasises that this is important useful for search engines and screenreaders for accessibility

strong tag tells that this text is important

The difference between strong & em is that "em" changes the meaning of the sentence subtlely, while strong tells that the text is important

"blockquote Tells the search engines that this page has a quote"

q tag is used for short quotes

abbr tag: WWF abbr tag is used for abbreviation.

definition tag:

A black hole is a region of space from which nothing, not even light, can escape.

del and ins tag: It was the worst best idea she ever had

s tag for StrikeThrough
